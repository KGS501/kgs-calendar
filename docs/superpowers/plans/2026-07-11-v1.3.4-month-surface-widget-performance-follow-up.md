# V.1.3.4 Month Surface And Widget Performance Follow-Up Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Hide inactive tasks from month surfaces, order app month pills by occurrence time, and make Month/Multi widget navigation atomic and effectively immediate for rapid taps.

**Architecture:** Put month-task visibility and app month-row ordering in small pure functions with unit coverage. Keep widget navigation revision-based, replace the partial acknowledgement with a complete cached page or complete skeleton, remove the fixed delay, and maintain a cancellable 13-month model-cache window.

**Tech Stack:** Kotlin, Android App Widgets/`RemoteViews`, Jetpack Compose, coroutines, JUnit 4, Gradle wrapper, ADB.

## Global Constraints

- Keep `versionCode = 14` and `versionName = "V.1.3.4"`.
- Completed and cancelled tasks are hidden only from app/widget month day cells; other views retain their existing settings.
- Existing multi-day lane, continuation, fade, and overflow behavior must not change.
- Existing task priority color, glow, motion, and scale behavior must not change.
- Cache lightweight `WidgetMonthPage` models, not complete `RemoteViews` trees.
- Every widget update must verify that its `MonthNavSnapshot.revision` is still current immediately before application.
- Run Gradle tasks sequentially with JDK 17 and the checked-in `gradlew.bat`.

## Planned File Structure

- `domain/model/MonthTaskVisibility.kt`: pure shared inactive-task rule for all month surfaces.
- `ui/month/MonthRowOrdering.kt`: pure comparator contract for app month-row candidates.
- `widget/WidgetMonthNavigation.kt`: navigation state plus initial-page and warm-window policy.
- `widget/WidgetMonthModel.kt`: neutral target-month loading skeleton model.
- `widget/WidgetMonthPageSource.kt`: 13-month cancellable cache warming.
- `widget/KgsWidgetProvider.kt`: atomic complete/skeleton navigation rendering with no artificial delay.
- Existing focused test files cover each pure policy before production integration.

```kotlin
package com.kgs.calendar.ui.month

import java.time.LocalDate

internal interface MonthRowOrderItem {
    val spanDays: Long
    val start: LocalDate
    val occurrenceSortMillis: Long
    val title: String
}

internal val MonthRowOrderComparator = Comparator<MonthRowOrderItem> { left, right ->
    right.spanDays.compareTo(left.spanDays)
        .takeIf { it != 0 }
        ?: left.start.compareTo(right.start).takeIf { it != 0 }
        ?: if (left.spanDays == 1L && right.spanDays == 1L) {
            left.occurrenceSortMillis.compareTo(right.occurrenceSortMillis).takeIf { it != 0 }
        } else {
            null
        }
        ?: left.title.compareTo(right.title)
}
```

```kotlin
internal enum class MonthNavigationPageStage { Complete, Skeleton }

internal data class MonthNavigationInitialPage(
    val page: WidgetMonthPage,
    val stage: MonthNavigationPageStage,

internal fun selectMonthNavigationInitialPage(
    cached: WidgetMonthPage?,
    skeleton: WidgetMonthPage,
): MonthNavigationInitialPage = if (cached != null) {
    MonthNavigationInitialPage(cached, MonthNavigationPageStage.Complete)
} else {
    MonthNavigationInitialPage(skeleton, MonthNavigationPageStage.Skeleton)
```

Add to `WidgetMonthModel.kt`:

```kotlin
internal fun WidgetMonthPage.loadingSkeleton(color: Int): WidgetMonthPage = copy(
    cells = cells.map { cell ->
        if (!cell.inCurrentMonth) {
            cell.copy(items = emptyList(), totalItemCount = 0)
        } else {
            val laneCount = if (cell.date.dayOfMonth % 4 == 0) 2 else 1
            val placeholders = (0 until laneCount).map { lane ->
                WidgetMonthItem(
                    id = "month-skeleton:${cell.date}:$lane",
                    title = "",
                    color = color,
                    sortMillis = Long.MAX_VALUE,
                    lane = lane,
                    continuesFromPrevious = false,
                    continuesToNext = false,
                    fadesFromPrevious = false,
                    fadesToNext = false,
                    completed = false,
                )
            }
            cell.copy(items = placeholders, totalItemCount = placeholders.size)
        }
    },
)
```

The renderer supplies `palette.muted` as the neutral color only when `hasCompleteData` is false. The skeleton is static; do not introduce a frame animation.
